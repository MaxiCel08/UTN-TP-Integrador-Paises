## Trabajo Practico Integrador: Programación I

### PAISES.CSV

## Instrucciones de Uso - Paises.CSV

El uso de este programa se dedica y se enfoca en manejar información basica de archivos .CSV con el uso de algoritmos
secuenciales, condicionales y repetitivos. El usuario puede usar este programa para manejar filas de información
conteniendo paises y sus respectivos atributos como su indice de población, superficie en metros cuadrados, y
el continente en donde reside.

El usuario tiene varios metodos disponibles para modificar el archivo y ver sus contenidos:

**Metodo 1**: Agregar un país. (El usuario ingresa un país con informacion personalizada y validada por el programa.)

**Metodo 2**: Actualizar los datos de un país. (El usuario puede elegir un país por nombre y modificar sus datos.)

**Metodo 3**: Buscar país por nombre. (El usuario puede buscar un país por nombre y ver la información.)

**Metodo 4**: Filtrar países por
Continente: Ver los países y sus respectivos continentes.
Rango de población: Definir un rango numerico para filtrar los datos de población.
Rango de superficie: Definir un rango numerico para filtrar los datos de superficie.

**Metodo 5**: Ordenar los países por
Orden alfabetico: Ver los nombres de países en orden alfabetico.
Orden menor a mayor: Ver los indices de población de orden ascendiente.
Orden de superficie: Ver los datos de superficie de los países en orden ascendiente y/o descendiente.

**Metodo 6**: Ver estadisticas
Paises extremos: Ver los países con mayores y menores indices de población.
Promedio superficie: Ver el promedio de superficie entre todos los países.
Paises y continentes: Ver una lista de cada continente con el número de países que marca.

**Metodo 7**: Salir del programa.
